Proyecto Inicio 2024

Este fue el proyecto con el que se inicio el aprendizaje en .NET 8 y Blazor Webassembly, realizado conjuntamente con el profesor de la materia modelado y arquitectura de software para un crud basico de tipos de documentos.

Como resultado logramos implementar conceptos como paginas razor, estilos css, AutoMapper, SweetAlert, entre otras tecnologias innovadoras para el desarrollo web.
