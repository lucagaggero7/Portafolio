GitHub Proyecto INNOBRA

En este proyecto grupal de 4 personas para la materia Práctica Profesionalizante I, desarrollamos una aplicación web de gestión de obras para computadoras, siguiendo los requisitos del profesor como cliente. Trabajamos con la metodología ágil Scrumban, dividiendo 2 casos de uso por integrante. Una vez completados la base de datos (SQL con Entity Framework), el servidor (ASP.NET 8) y las páginas para los CRUD de cada entidad, integramos todo en solo 2 componentes Razor Pages.
