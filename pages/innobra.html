Proyecto INNOBRA

Este proyecto particular es un trabajo grupal de 4 personas para la materia Practica Profesionalizante I en la cual teniamos como cliente al profesor y segun sus requisitos teniamos que desarrollar una aplicacion web de gestion de obras espesificamente para computadoras.

Como resultado logramos trabajar mediante la metodologia agil Scrumban y dividir los casos de uso en 2 por cada integrante para una vez finalizados, tanto la base de datos sql con entity framework, el servidor con los controllers y las page que harian los crud de cada entidad, integramos todos los casos de uso para que quedaran reducidos a tan solo 2 componentes razor pages.
