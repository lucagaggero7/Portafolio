GitHub Proyecto Final de Programacion II

GitHub Final Project: Programming II

Esto es un proyecto para una tienda de electronicos en WinForms como trabajo final de la materia programacion II en el cual creamos una solucion con 4 capas en .NET Framework 4.8, Frontend de WinForms, y Base de datos en Microsoft Access con login, register, roles basicos, crud de productos, filtrados, carrito de compras, factura electronica, etc.. Tambien hay un reporte de las ventas facturadas en PowerBI (.pbix)

This is a project for an electronics store in WinForms as the final project for the Programming II course, in which we created a solution with 4 layers in .NET Framework 4.8, a WinForms frontend, and a database in Microsoft Access. It includes features such as login, registration, basic roles, product CRUD, filtering, shopping cart, electronic invoicing, etc. There is also a sales report in PowerBI (.pbix).
