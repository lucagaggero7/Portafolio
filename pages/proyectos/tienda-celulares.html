GitHub Proyecto Tienda de Celulares

GitHub Electronics Store Project

Esto es una pagina web de tienda de celulares en Angular como proyecto personal en el cual creé una web api usando Visual Studio 2022 con la plantilla de ASP.NET Web Api en su version .NET 9 , frontend en Angular 19 como proyecto separado, y base de datos en SQL Server utilizando Entity Framework.

This is a mobile phone store website as a personal project, where I created a web API using Visual Studio 2022 with the ASP.NET Web API template in .NET 9, the frontend in Angular 19 as a separate project, and a database in SQL Server using Entity Framework.
