GitHub Zetta Services

El proyecto es parte de la asignatura Modelado y Arquitectura de Software de la carrera de técnico superior. El objetivo es automatizar las tareas de Víctor, un profesional de plomería, electricidad y calefacción, que anteriormente cargaba datos de clientes y obras en un Excel. Se desarrolló una base de datos con un enfoque database-first usando .NET 8 y SQL, y se implementó un frontend en Blazor WebAssembly. Aprendi a hacer mi primer deploy para publicar el sistema con despliegue e integración continua.

The project is part of the Software Modeling and Architecture course in the higher technical program. The objective is to automate the tasks of Víctor, a professional in plumbing, electricity, and heating, who previously entered client and project data into an Excel spreadsheet. A database was developed using a database-first approach with .NET 8 and SQL, and a frontend was implemented in Blazor WebAssembly. I learned to perform my first deployment to publish the system with continuous integration and deployment.
