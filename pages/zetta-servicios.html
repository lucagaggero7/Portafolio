GitHub Zetta Servicios

El proyecto es parte de la asignatura Modelado y Arquitectura de Software de la carrera de técnico superior. El objetivo es automatizar las tareas de Víctor, un profesional de plomería, electricidad y calefacción, que anteriormente cargaba datos de clientes y obras en un Excel. Se desarrolló una base de datos con un enfoque database-first usando .NET 8 y SQL, y se implementó un frontend en Blazor WebAssembly. Aprendi a hacer mi primer deploy para publicar el sistema con despliegue e integración continua.
